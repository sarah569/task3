1- open xampp or any program like it
2- open phpmyadmin
3- create database and name it 'task3' in phpmyadmin without password
4- put 'task3' folder in xampp->htdoc
5- open 'task3' folder in any text editor program
6- run the project
7- choose file 'schema.sql' from folder 'sql' and press 'Import Database' to import schema to database
8- choose file 'reset.sql' from folder 'sql' and press 'Reset Database' to reset database
9- choose file 'Task 3 CSV.csv' from folder 'Template' and press 'import' to import CSV file to database and display it in browser
